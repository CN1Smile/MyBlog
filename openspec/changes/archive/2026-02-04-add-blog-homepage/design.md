## Context

这是一个全新的博客项目，需要从零开始构建。我们选择纯前端技术栈以保持简单，使用原生 HTML、CSS 和 JavaScript，无需构建工具。

## Goals / Non-Goals

**Goals:**
- 创建现代化、响应式的博客首页
- 保持代码简洁，易于理解和扩展
- 确保良好的用户体验（加载状态、流畅动画）

**Non-Goals:**
- 不实现后端 API（使用 JSON 文件模拟）
- 不实现文章详情页（后续变更处理）
- 不实现用户认证功能

## Decisions

### Decision 1: 使用原生技术栈

使用原生 HTML5、CSS3 和 ES6+ JavaScript，不依赖框架。

**理由：** 对于简单的博客首页，框架会增加不必要的复杂性。原生技术更轻量，加载更快。

### Decision 2: CSS Grid 布局

使用 CSS Grid 实现响应式文章卡片布局。

**理由：** Grid 提供强大的二维布局能力，代码简洁，浏览器支持良好。

### Decision 3: JSON 文件作为数据源

使用静态 JSON 文件存储文章数据，通过 fetch API 加载。

**理由：** 允许我们专注于前端实现，同时保持与真实 API 调用相同的异步加载模式。
